# Heart Disease Prediction

This project predicts heart disease using machine learning.  
The UCI Heart Disease dataset was used.

## Models
- Logistic Regression – Accuracy: 82%
- Decision Tree – Accuracy: 75%

Logistic Regression performed better.

## How to run
1. Install requirements: `pip install -r requirements.txt`
2. Open the notebook: `jupyter notebook notebooks/01_EDA_and_baseline.ipynb`
3. Run all cells.

## Files
- `data/heart.csv` – dataset
- `notebooks/01_EDA_and_baseline.ipynb` – code
- `requirements.txt` – dependencies
